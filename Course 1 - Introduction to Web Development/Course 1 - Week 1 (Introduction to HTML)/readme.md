Welcome to the course! In this module, you will start by downloading and installing the code editor **Visual Studio Code**. You will then learn to identify different parts of the HTML language, manually add basic tags for web pages, and correctly use opening and closing tags with content to markup HTML files. You will also test your files for errors using a **HTML validator**.  

### **Learning Objectives**

By the end of this module, you will be able to:

- Download and install a code editor such as **Visual Studio Code** on your computer  
- Use basic **opening and closing tags** with content to markup HTML files  
- Manually add essential HTML tags, including:
  - `<!DOCTYPE html>`  
  - `<html>`  
  - `<head>`  
  - `<body>`  
  - `<title>`  
  - `<meta charset="utf-8">` inside the `<head>` tag  
- Test your HTML files for errors using the [W3C HTML Validator](https://validator.w3.org/)  
- Identify different parts of the HTML language, such as **tags** and **attributes**  

This module provides a solid foundation for building static web pages and understanding the structure of HTML.
